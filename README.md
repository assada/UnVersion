# UnVersion

Simple Spigot plugin that replaces versions in the **server list** and **F3 menu**

## Requirements
* Spigot\Paper 1.18.2, 1.19
* ProtocolLib 4.8+ (for 1.19 use ProtocolLib 5.0+)


## Configuration
```yaml
version:
  serverList: Dead 1.19
  f3: '&l&6DeadGuru&r'
```
